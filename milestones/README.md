# Course Milestones Archive

This folder contains the milestone scripts used during the robotics course progression that led to the final integrated manipulator project.

- milestone-02: Forward kinematics and ROS command publishing foundations.
- milestone-03: Jacobian, inverse kinematics, and velocity kinematics validation.
- milestone-04: Task-space and joint-space trajectory planning experiments.

These files are preserved as learning artifacts and are kept separate from the production simulation and hardware directories.
